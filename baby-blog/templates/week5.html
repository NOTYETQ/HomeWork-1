Week 5 - Logging & Flags

This week, we covered flags and logging in Go, including how to use log and slog for debugging applications.

Using Logs in Go

Logs help us track application behavior and debug errors efficiently.

// Logging example in Go
package main

import (
    "log"
    "os"
)

func main() {
    file, err := os.OpenFile("app.log", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
    if err != nil {
        log.Fatal(err)
    }
    defer file.Close()

    logger := log.New(file, "INFO: ", log.Ldate|log.Ltime|log.Lshortfile)
    logger.Println("Server started")
}
