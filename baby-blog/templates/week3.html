Week 3 - Self-Assessment & Git

This week, we had a self-assessment that involved creating a file directory, setting up Git, writing a simple Go program, and pushing it to GitHub.

Setting up a Repository

We initialized a Git repository, created a simple web server in Go, and pushed it to GitHub.

// Initializing a new Git repository
$ git init
$ git add .
$ git commit -m "Initial commit"
$ git remote add origin https://github.com/user/repository.git
$ git push -u origin main
