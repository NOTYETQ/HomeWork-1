Week 3 - Self-Assessment & Git

This week, we continued with handlers and went more into depth about how they worked. we also had a self-assessment that involved creating a file directory, setting up Git, writing a simple Go program and some handlers for our webserver, and pushing it to GitHub. This was done to test everything that we had learned so far in class
